overlayColor
Android
​

当图片有圆角的时候，指定一个颜色用于填充圆角处的空白。虽然一般情况下圆角处是透明的，但在某些情况下，Android 并不支持圆角透明，比如：

  • 某些 resize 模式比如'contain'
  • GIF 动画

常见的用法就是在不能圆角透明时，设置overlayColor和背景色一致。

详细说明可参考https://frescolib.org/docs/rounded-corners-and-circles.html。

类型
string
